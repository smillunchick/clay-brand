# Clay · כלי

**Sacred in the everyday.**

Israeli pottery studio making traditional and contemporary ceramics. The name is a bilingual pun: "Clay" (the material) / "כלי" (kli — vessel) / "כלי קודש" (klei kodesh — holy vessels).

---

## Brand at a Glance

**Personality:** Warm · Sincere · Down-to-earth

**Colors:**

| | Terracotta | Cream | Glaze |
|---|---|---|---|
| Primary | `#A25D39` | `#F2EFE4` | `#4A6B7A` |
| Dark | `#813F25` | `#EDE8D8` | — |
| Light | `#B6704E` | `#F5F3EC` | `#6B8D9C` |

**Typography:**
- Display: [Cormorant](https://fonts.google.com/specimen/Cormorant) (Light 300, Regular 400, Italic)
- Body: [DM Sans](https://fonts.google.com/specimen/DM+Sans) (Light 300, Regular 400, Medium 500)

**Logo:** Hebrew-influenced calligraphic wordmark. The angular strokes reference Hebrew script — the bilingual meaning lives in the letterforms themselves.

**Voice:** Warm but direct. Speaks from the workshop, not the gallery. "Handmade in Israel" not "artisanally crafted ceramic experiences."

## Quick Start

1. Open `mockups/brand-system.html` in a browser to see the full visual system
2. Reference `design-tokens.css` for all color, type, and spacing values
3. Read `brand-guide.md` for usage rules, voice guidelines, and application notes

## Logo Usage

- **Primary:** terracotta on cream
- **Reversed:** cream on terracotta or dark backgrounds
- **Minimum size:** 32px width (signature dots become illegible below this)
- **Counter fill:** the 'a' counter must match the background color
